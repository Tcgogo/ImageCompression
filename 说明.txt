## 安装 node
1. 下载软件 `https://nodejs.org/dist/v12.19.0/node-v12.19.0.pkg`
2. 一直下一步
3. 点击 `install.bat`

## 使用
1. 把要压缩的图片放入 `input` 文件夹中
2. 点击 `build.bat`
3. 压缩后的图片在 `output`

## 图片压缩
- 白嫖于 `tinypng` 压缩
- 大小有限制
- 数量没限制
- 把压缩的的图片放到 `input` 中
- 只支持 `tinypng` 格式

- 装依赖
```sh
yarn 
# or
npm i
```
- 压缩
```sh
yarn build
# or
npm run build
```
